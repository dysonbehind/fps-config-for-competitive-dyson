> **fps config for competitive dyson** — это глубоко оптимизированная конфигурация движка Source 2, которая полностью перестраивает рендеринг, работу шейдеров, потоковую загрузку текстур, систему частиц и многопоточность на низком уровне.

## ⚙️ Установка (пошагово)

1. Скачай все файлы из актуального релиза  
2. Сделай резервную копию оригинальных `gameinfo.gi` и `video.cfg`
3. Замени файлы на версии из fps config for competitive dyson  
4. **Обязательно** отредактируй `video.cfg`:  
   - Вставь свои `VendorID` и `DeviceID`  
   - Укажи своё разрешение экрана  
5. Поставь оба файла в режим **«Только чтение»**  

> ⚠️ После каждого крупного обновления Deadlock проверяй файлы — игра может их перезаписать.
